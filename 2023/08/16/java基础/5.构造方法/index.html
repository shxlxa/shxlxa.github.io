构造方法

发表于2023-08-16|更新于2024-03-04|java
|阅读量:

java中所有的对象都是new出来的，所有对象的内存都是在堆空间，所有

保存对象的变量都是引用类型。

1.构造方法constructor

构造方法，也叫构造器，能够更方便地创建一个对象，其特点：

  • 方法名必须与类名一样
  • 没有返回值类型
  • 可以重置

this是指向当前对象的引用
