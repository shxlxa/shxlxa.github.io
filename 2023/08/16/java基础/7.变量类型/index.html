变量类型

发表于2023-08-16|更新于2024-03-04|java
|阅读量:

static常用来修饰类的成员：成员变量、方法、嵌套类

成员变量

被static修饰的变量叫类变量、静态变量，其特点：
  • 在程序运行过程中只占用一份固定的内存（存储在方法区）
  • 可以通过类访问、实例访问（不推荐）
没有被static修饰，叫实例变量
  • 每个实例都有一份内存
  • 只能通过实例访问，不可以通过类访问
局部变量

  • 类在方法中的变量。局部变量没有默认值。所以局部变量被声明后，必须经过初始化，才可以使用。

  • 访问修饰符不能用于局部变量；

方法

被static修饰，叫类方法、静态方法
  • 可以通过实例、类访问
  • 内部不可以使用this，因为this代表对象，而静态在时，有可能没有对象，所以this无法使用
  • 可以直接访问类变量、类方法
  • 不可以直接访问实例变量、实例方法
没有被static修饰，叫实例方法
  • 只能通过实例访问
  • 内部可以使用this
  • 可以直接访问实例变量、实例方法
  • 可以直接访问类变量、类方法

静态导入

使用了静态导入后，就可以省略类名来访问静态成员

导入静态成员 import static com.cft.Person.age;

单例模式

如果一个类设计成单例模式，那么在程序运行过程中，这个类只能创建一个实例

final

  • 被final修饰的类，不能被子类化，不能被继承
  • 被final修饰的方法，不能被重写
  • 被final修饰的变量，只能进行1次赋值
